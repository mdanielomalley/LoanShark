Loan Shark

A Coding Challenge

Loan Shark is a coding challenge built to display every number between 0 and 100 while printing multiples of the fizz value as "Fizz", and multiples of the Buzz value as "Buzz". When the two values share a multiple, the program prints "Fizz-Buzz."

Instructions:

Enter a starting value and an ending value. Press the button and the values will appear.
